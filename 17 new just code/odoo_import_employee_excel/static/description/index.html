Import Employees from Excel

Import Employees with Image in Odoo using Excel

This Odoo app allows you to Import Employees from Excel files.

This module allows you to import Employees and their images using URL/Path from excel.

Make sure you have xlrd Python library installed.


Main Features:

  • Allow you to Import Employees with columns listed below along with employee image.
  • Allow to import *.jpg or .png or .jpeg ... files types.
  • For more details you can see the screenshots below and watch the video.
  • Allow you to have an image file path of your directory and Website URL where you keep your image. (Both options can work together with single import.)
  • Allow you to give http or https as website URL to import employee image using Excel.
