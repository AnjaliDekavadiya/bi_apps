Return Material Authorization Management

Repair - Scrap - Replacement

This module allow your customer to create Return Material Authorization from portal and allow you to manage Repair, Scrap, Replacement

Main Features:

- Your customer can go to Sales order in Portal My Account and Create RMA request.
- Allow your sales team to manage Return Material Authorization request in backend.
- Before confirm allow your sales team to select Method on Return Material Authorization Lines.
- Allow sales manager to Approve Return Material Authorization confirmed request.
- Allow team to create repair order directly from Return Material Authorization form.
- Allow team to create scrap directly from Return Material Authorization form.
- Allow team to create replacement order (Sales order with zero price) directly from Return Material Authorization form.
- Print Return Material Authorization.
- Send by email Return Material Authorization to customer.
- Sales order form has reference of All Return Material Authorization orders.
- Customer form has reference of all Return Material Authorization orders.
- For more details please see video.
