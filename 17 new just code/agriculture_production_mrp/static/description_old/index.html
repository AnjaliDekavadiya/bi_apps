Agriculture Integration with Manufacturing (MRP)

Agriculture Manufacturing Integration in Odoo

This app makes Agriculture module Integration with Manufacturing (MRP) and Bill of Materials. So using this module farm management / agriculture module will be linked with MRP. So you can use the Odoo MRP module extensively with Agriculture processes.


Main Features:
  • Allow you to Create Bills of Material from Crop.
  • Bills of Material will be linked with Crop.
  • Allow you to Create Manufacturing Order from Crop Request.
  • For more details please check below screenshots and watch the video.
