Odoo Marketplace Seller Store Pickup

Let the customers pick up their Orders directly from the sellers Store!

Odoo Marketplace Seller Store Pickup allows you to enable the option for allowing your customers to pick up their orders directly from the store. It is a way to offer the convenience to your customers so that they can pick the order in the preferred time and day.

Introduce Buy Online, Pick Up From Store Strategy on your Odoo Marketplace!!

The module allows customers to pick up their orders from the seller store physically using Odoo Marketplace Seller Store Pickup Module.

It adds store pickup as one of the delivery options on the website. The seller can create multiple store locations and add them to the website for the customers to choose from while opting for store pick-up. Various details for customers like address, contact number etc can be added to these stores. Various details for customers like address, contact number etc can be added to these stores.

Note: Odoo Marketplace Seller Store Pickup Module is a dependent extension on the following modules:

  • Odoo Multi-Vendor Marketplace
  • Odoo Marketplace Shipping Per Product
  • Odoo Shipping Per Product

These are the framework modules to provide the structure for Odoo Marketplace Seller Store Pickup Module. The base modules need to be installed first for Odoo Marketplace Seller Store Pickup Module to work.

Detailed Features List

Enable store-pickup option for order deliveries

  • The module facilitates adding Store pickup as the option for order delivery on the Odoo website.
  • It allows the customers to place the order on the Odoo marketplace and then physically pick up their order with the store.

Add different physical store locations for customers

  • The seller can add various physical store locations with details such as address, contact number, store timings, etc. for the customers.

Allows customers to choose the delivery preferences as per their wish

  • The customers can choose the date, time and store location for the pick.
  • Customers can choose the nearby location as per their convenience.

Manage the store pickup delivery from the Odoo backend

  • The module lets you view the chosen location for pickup by the customers.
  • You can even track the delivery type in the order lines in the Odoo backend.
