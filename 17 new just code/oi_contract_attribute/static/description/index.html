Employee Contract Attributes

"Make Sure Everyone Gets What They Deserve"

This module allows defining specific attributes for employee contracts for a particular period.

Description

Attributes of the contract of employment are details that should be in the contract of employment to adequately define what is expected of the employee and the employer.

The “Employee Contract Attributes” are used to define a set of attributes for a contract. Attributes are used to determine which feature value should be available for a specific Employee or a Batch of Contracts and Department for a specific period. Based on this definition, the attributes are displayed as tabs on the Contract Form. Also, the “Employee Contract Attributes” module helps to define whether the specific case is related to the employment contract or not.

Key Features

  • Supporting multi-level manager approvals.

  • Highly flexible in defining different periods.

  • High flexibility to define different attributes.

  • Ability to provide attributes at different level options.

  • Great configuration feature to define any needed attribute based on contract specifications.

  • Fully integrated with Odoo applications, especially with Employee, Payroll, and Accounting.

  • Ability to search attributes with different dimensions, for example (employee, status, value, tags, contract, job position, etc.).
