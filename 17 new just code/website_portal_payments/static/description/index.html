Payment Portal for Customers and Vendors

This app allows customers and suppliers to login in My Account Portal of your website and view list payments and allow them to print receipts PDF format as shown in below screenshots.


Main Features:
  • This module shows Account *Payment* Receipts on My Account Portal to your customers and vendors using portal login.
  • Allow customers and suppliers to login in My Account Portal and view Payments and Print Receipts in PDF format.
  • For more details please watch the video and check below screenshots.
