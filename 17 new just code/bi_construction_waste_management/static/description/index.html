Waste Management for Construction and Recycle Process for Contracting Odoo Apps

Recycle Waste management for Job Contracting Construction Odoo Apps allow user to manage Construction Waste Materials using scrap and reuse material for your Construction and Job Contracting business. After installing this module waste material tab will be available on job costing sheet and Construction user can able to add all the wasted materials here and based on that scarp order and picking will be automatically generated based on configuration for that specific job order.

Features

Waste packages

Allow job order user to add Waste Materials on Job Order form and based on that Create Internal Picking or Stock Scrap based on Waste Method selected.

Show scrap and picking in smart button

Using create button on Waste material tab aumatically create scrap record or internal picking record based on method selected.

Reuse or Scrap

User can decide whether waste will be reuse or scrap.

List out waste packages

Show all waste packages list.
