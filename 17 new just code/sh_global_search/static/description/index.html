Global Search

A global search is used to search any object based on the configuration. You can search all object's data easily. You can also configure one to many fields. The "Global Search" is visible to all odoo users. The search box is available at the top of the menu. Search results on click redirect to that record on the new tab. You can easily perform a search on multi-company objects also we have beautifully shown company names before so you can see search results related to that company, we have shown object (model) nicely whenever you have a search query found in multiple objects, you can very easily see the difference of different objects. It also takes care of access rights of users, if a user doesn't have any object access then it will not show that record in search results. if the user has multi-company enabled then it shows multi-company results. We have made this look fully configurable you can easily configure what is an important object(model) and the related fields of that object(model), We have made security groups for these configurations so only those who have the right that user only can configure this global search objects and fields, All internal users can use this global search feature.
