Helpdesk Ticket Working Hours and Upcoming Holidays in Odoo

Working Hours Show on Helpdesk Ticket Form

Employee Working Hours on Support Ticket Form

This module sets a default working hours/time on Helpdesk Team & show support working hours & upcoming holidays on website helpdesks ticket form so your customer(s) can see support working hours & upcoming holidays during website support ticket submit form.


Main Features:

  • The Support Manager can set default working hours/time on the team under the configuration menu of the helpdesk support ticket app in the backend.
  • Customers can see support working hours on the Website Helpdesks Support Ticket submit form as shown below.
  • Customers can see support for upcoming holidays on Website Helpdesks Support Ticket submit form.
  • For more details please see below screenshots and watch the video.
  • ONLY COMPATIBLE FOR ODOO COMMUNITY EDITION.
