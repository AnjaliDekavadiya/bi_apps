Import Expense and Expense Sheet from Excel

Expense and Expense Sheet Import from Excel Odoo

This app allow you to Import Expense and Expense Sheet from excel file.

Main Features:

  • Allow you to import multiple Expense from excel.

  • Allow you to Import Expenses.

  • Allow you to Import Expense Sheets.

  • Sample Excel files are available inside sample folder in app.

  • This module allow you to Import Employee Expenses and Expense Sheets from Wizard using Excel Import.

  • Make sure you have xlrd Python library installed.

  • For more details see video and below screenshots.
