Noteworthy Points

  • You need to add multiple images while creating a product from the odoo backend so that it shows up like a 360 rotating view.
  • Sequence number of the image is very important.
  • Images will spin in the sequence according to sequence number in order to have a smooth 360 rotating view.
  • Images should be arranged in a sequential manner.
  • For a good result of a 360 view, use at least 25 images.
