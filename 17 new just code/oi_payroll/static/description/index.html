The Standard Payroll Management Module for Odoo 14

Payroll Management

"Manage Your Employee Payroll Records"

This module allows you to manage employee salaries in your organization.

Description

“Payroll management” is the management of employee salaries in an organization; also, it includes timely issuance of payslips, allowances, deductions, and so on. “Payroll” module helps you to keep track of payslips and create corresponding journal entries by allowing you to manage employee salaries, configure salary rules, and configure salary structures. The module will enable you to generate monthly payslips entries automatically and perform reports on created entries. “Payroll Management” also provides payslips batches, which allows you to include one or more employees or category batches. The mass creation of payslips for a group of employees can be achieved through this scenario.

Key Features

  • Ability to register contributions.

  • Ability to generate payslips batches.

  • High flexibility in defining payslips batch types.

  • Easy traceability for payslips by different dimensions.

  • Multilevel configurable approval and rejection notification.

  • Ability to create payslips manually as well as automatically.

  • Managing complicated workflow approvals, based on organization policies.

  • Great discretion in creating salary rules, salary rule categories, and structures.

  • Fully integrated with other Odoo applications, especially with Employee, Accounting, Leave, etc.
