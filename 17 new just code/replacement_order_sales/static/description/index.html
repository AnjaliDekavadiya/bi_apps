Replacement Order for Sales Process

Sales Replacement Order in Odoo

This app allows your sales team to create a replacement order for sales orders.

Main Features:

  • Allow your sales manager to configure the replacement reasons.
  • Allow your sales team to create a replacement order with a select replacement reason and original sales order.
  • Allow you to view or display the replacement order from the sales order as a smart button
  • Unique replacement order number generation.
  • Allow you to display the replacement order details on the pdf report.
  • Allow your sales manager or sales team user to group by the sales analysis report with original sales order and replacement reason.
  • For more details please check below screenshots and watch the video.

Menus Available:

  • Sales
    • Orders
      • Replacement Orders
    • Configuration
      • Replacement Reasons
