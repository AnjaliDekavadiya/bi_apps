Why Marketplace Voucher

The Odoo Marketplace Voucher module is designed to boost customer engagement and sales in eCommerce businesses. It allows businesses to create attractive vouchers and coupons for marketplace products, particularly during festivals. The module's versatility allows sellers to participate seamlessly, fostering a collaborative environment and driving marketplace growth. It also builds customer interest, ensuring continued engagement and repeat visits. The module generates online traffic, contributing to the marketplace's visibility and popularity, ultimately positioning the platform for sustained success and profitability. .

Benefits of Marketplace Voucher

  • Marketplace Voucher is used to create vouchers or coupons for the products of Marketplace.
  • Admin can manage the seller coupons.
  • Allow sellers of Marketplace to generate a coupon code for their own products.
  • A seller can generate single or multiple coupon codes at a time.
  • Sellers can specify the validity of the vouchers.
  • A seller can check the history of its own coupon.
  • A seller can set the coupon value and number of available coupons for all the customers.
  • The buyer of the marketplace can use coupon discount.
  • Buyer can add coupon value from a different seller in the seller's product.
  • A seller can specify the product on which the created voucher will be applicable.
  • A seller can check the Credit and Debit of Vouchers using history.

Seller View of Marketplace Voucher

Gift Voucher menu is visible to the admin, admin can create the voucher.

Admin can enter the details of the voucher.

Admin can add any product of any seller of marketplace in the voucher if voucher is for specific products.

Admin can check the history of all the vouchers of marketplace in Voucher History menu.

Admin can also see the detailed view of all the vouchers in the form view.

Detailed Features List

Below is the list of features odoo marketplace voucher

Add Discount Coupons

  • Each coupon has a unique code.
  • The customer needs to use the code at the checkout page to avail of the discount.

Allows To Create Odoo Marketplace Discount Vouchers

  • The seller can create coupons for his products and the admin can create coupons for all the sellers.
  • Customers can use the digital coupon codes while making purchases from Marketplace.

Create Marketplace Coupons

  • The seller can create a coupon for all his products or specific products.
  • The admin can create coupons for all seller's products or specific products

Set Minimum Cart Amount Eligibility For Coupon Use

  • Set a minimum cart amount for the customer to avail of the coupon discount.
  • Each Odoo Marketplace coupon can have a different minimum cart amount.

Limit The Number Of Usage And Validity Of Each Coupon

  • Allows you to enter the number of times a coupon can be used on the Odoo marketplace.
  • Once the number is reached the coupon automatically expires.

Shows Available Coupons On The Marketplace Checkout Page

  • The customers can see the available coupons on the checkout page.
  • If a customer's cart has products from multiple sellers offering coupons, the customer can redeem coupons for only one seller at a time.

See Usage History Of Marketplace Coupons In Odoo

  • The admin can see the usage history of all the coupons from all the sellers.
  • The sellers can see the history of usage for their coupons only.
