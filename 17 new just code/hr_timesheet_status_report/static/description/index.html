Employee Missing Timesheet Reports Odoo App

This app allows your timesheet manager to print employee missing timesheets in PDF format as shown in below sample.

Main Features:

  • This module allows a timesheet manager to print a timesheet report for incomplete timesheets of employees.
  • Defined Reports: Print Timesheet Status as wizard to print in pdf format as shown.
  • Also the app is having a cron job run automatically weekly and send email to listed persons/managers on company form for employees incomplete timesheet report in PDF.
  • Apart from a cron job, the system also allows you to print the same incomplete timesheets report using wizard as shown in the screenshots below.
  • For more details please watch the video and check below screenshots.
