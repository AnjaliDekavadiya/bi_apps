Event Management for Event Organiser

Job Costing Sheet, Material Requisition, Sales Estimate for Event Organizer / Specialists for Event Management

This app allows any event organizer/event coordinator/event planner company to manage their customer events with a job costing sheet, estimation, material requisition, and meetings.



Main Features:
  • Event:Allow you to set customer and project on event form (Odoo standard).
  • Allow your event team to view the project (Odoo Standard) from the event (Odoo Standard).
  • Allow your event team to view the project tasks (Odoo Standard) from the event (Odoo Standard).
  • Sales User and Event Manager can view the Sales Quotations (Odoo Standard) from the event (Odoo Standard).
  • Event Manager can view the Material Requisitions from event.
  • Event Manager can view the Job Costsheet from event.
  • Purchase User and Event Manager can view the Request for Quotations (Odoo Standard) from the event(Odoo Standard).
  • Billing User and Event Manager can view the Customer Invoice (Odoo Standard) and Vendor Bills (Odoo Standard).
  • Meeting: Allow your users to schedule and show meetings (Odoo Standard) for event.
  • Job Costsheet: Allow project manager to create and manage job costing sheet for event and based on that they can planned cost for material, labour and overheads cost.
  • Sale Estimate: Allow your event team or sales team to create and manage sales estimates for events and send estimates to the customer for cost.
  • Sales Quotation: Allow your event team or sales team to create and manage sales quotations and sales orders (Odoo Standard) for events and send them to the customer for cost.
  • Customer Invoice: Allow your event team or invoice team to create and manage customer invoices (Odoo Standard) for events and send them to the customer.
  • RFQ / Purchase Order: Allow your event team or purchase team to create and manage purchase quotations and purchase orders (Odoo Standard) for events and send them to the vendor.
  • Vendor Bill: Allow your event team or invoice team to create and manage vendor invoices (Odoo Standard) for events and send them to the vendors.
  • Project Task: Allow your project team/event team users to create project tasks and also allow them to add and manage a checklist for the task related to event management as an event organizers.
  • Checklist Types: Allow you to configure checklist types and checklist templates as shown.
