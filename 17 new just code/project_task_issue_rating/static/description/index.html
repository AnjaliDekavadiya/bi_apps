Project Task Review with Rating by Customer.

Project Task Customer Review and Ratings Odoo App.

Customer Feedback and Rating for Tasks.

This app allows your project managers to send review requests to your customer for task work by project users. We have added a button on the task form which will allow them to send a review request to the customer by email using wizard.

Main Features:

  • Customers can give review and rating of tasks requested by the project team as shown.
  • Project managers can send a review request by running wizard on task form as shown.
  • For more details please see below screenshots and watch the video.

Edition Compatibility:
