Ticket Email Auto Notification

Helpdesk Support Tickets by Stages

THIS APP ONLY SUPPORT ODOO COMMUNITY EDITION

This app allow you to configure email notification when ticket stage changes.

Main Features
* Auto send email notification to the customer, followers, internal users / managers by selecting setting from the Helpdesk stages configuration.
* Allow you to set email template to send e-mail available Helpdesk Stages configuration.
* Allow you to have different email templates with your own email body and you can configure on stage form.
* THIS APP ONLY SUPPORT ODOO COMMUNITY EDITION.

Helpdesk Support Satge - Configuration

Check box "Need Auto Response" for auto email notification.

Select Custom Email Template formate to send mail.

Check box if you want to send email to Customer if ticket enter in that stage.

Check box for Followers Of Document to send email notification to the Followers of the support ticket.

Select Internal Users for send mail to the selected Internal users on the stage configuration
