Module to set an POS authorization Discount

for using discounts from POS UI.

In order to use this module correctly, you must access to: Point of Sale application. Then open the POS configuration screen to set the configuration, as follows:

In OTP Discount value we have a three value

(-1)
if admin does not need any discount.
(0)
if you need to accept any discount without admin permission.
(ex: 12%)
when we create order discount with this value or above we will send message with discount information to admin.
