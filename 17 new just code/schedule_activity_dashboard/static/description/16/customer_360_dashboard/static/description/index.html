Customer 360 Dashboard

Dashboard for Customers Odoo App

Customer Dashboard App

This app adds a customer 360 dashboard under sales application.


Main Features:

  • This module will allow your users to view the customer dashboard.
  • You can see the customer image, name, address, email on the dashboard.
  • You can see the sales button on Dashboard and while clicking on the sales button which shows customer sale orders.
  • You can see the meetings button on the dashboard and while clicking the meetings button which shows customer meetings.
  • You can see the CRM pipelines/opportunities button on the dashboard and while clicking the opportunities button which shows Customer opportunities.
  • You can see the invoices button on the dashboard and while clicking the invoices button which shows customer invoices.
  • You can see the analytic button on the dashboard and while clicking the analytic button which shows customer analytic.
  • You can see the payment button on the dashboard and while clicking the payment button which shows customer payment.
  • You can see the tasks button on the dashboard and while clicking the tasks Button which shows customer tasks.
  • You can see the project button on the dashboard and while clicking the project button which shows customer projects.
  • For more details watch the video and check below screenshots.
