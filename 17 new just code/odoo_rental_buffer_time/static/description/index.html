Product Rental Buffer Timings - Odoo App

Rental Buffer Start Time and Buffer End Time for rental products. Mainly used in case if you are renting items and before renting any item if you want to for example wash/clean products or if you want to reserve some time for previous customer return gap.. Same for buffer end time if you want to reservice product for next customer untill previous customer returns it and then you do some processing like cleaning, washing and ready for next order.

Main Features:

  • Allow you to configure Buffer Start Time and Buffer End Time in minutes/hours on product form.
    • If you set for example Buffer Start Time as 30 mins then your Reservation Backup Start Date Time will be reserve till 30 mins.
    • If you set for example Buffer End Time as 30 mins then your Reservation Backup End Date Time will be reserve till 30 mins.
  • Auto compute Buffer Start Datetime and Buffer End Datetime on sales order line and Buffer Time History.
