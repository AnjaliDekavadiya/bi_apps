What is necessary to run a smooth Amazon FBM business?

  • Import of Amazon orders securely
  • Sync stock for all or single product
  • Update order status
  • Log your data


As the native Odoo Amazon connector is only able to import sales orders (Amazon Connector Documentation) and has some major "challenges", this might not be enough for E-commerce businesses to succeed. We (4EGrowth GmbH) run our own Amazon business in Germany since 2010 and are aware of all important features when it comes to connecting Odoo with Amazon. We built our features on Odoo's native "solution" and made it usable for every Amazon Seller.





This 4EG FBM Amazon core extension provides

1) Import data at a specific time (for example if you missed/failed orders for any reason you can reimport). Pls note that Amazon has currently a threshold of ~ 40 orders.
2) Super important! Sync your odoo stock to Amazon. With native Odoo Amazon connector you have to do that manually! Just create a cron to update with our extension. (see also picture below 3.)
3) Update orders status manually or by cron. If for any reason you want to update your shipped orders again or if something went wrong
4) Log your data. Super important to identify problems with import/export. Furthermore you will see all imported sales order data of every sales order in sales order view. For example if some address data or tax is not set appropriate. (Only visible in debug mode)

Add on for a better Amazon business

5) Set your unique debitor account per marketplace instance. To get a better overview in accounting you might want to see on which marketplace you are having the profit. (see screenshot below 8.)
6) Set a sales team per marketplace. Get a quick overview wich marketplace is doing best in sales and compare over product categories/products/countries and more. Compare different times to see increase/decrease of sales per marketplace. (see screenshot below 8.)
7) Set a Warehouse per Marketplace. If you want to for any reason sell different stocks on different Marketplaces. (see screenshot below 8.)
8) Set a pricelist per marketplace (prices are not synced so far)
9) Having easy access menu on sales.
10) The marketplace (country) is set for each Amazon order
