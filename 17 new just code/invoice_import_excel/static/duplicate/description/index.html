Import Customer Invoice and Vendor Bill from Excel

This module will allow you to import invoice from excel. See video in Live Preview for more details.

Make sure you have xlwt Python module installed.

Following fields can be Import.

  • PRODUCT
  • DESCRIPTION
  • QUANTITY
  • UNIT OF MEASURE
  • UNIT PRICE
  • DISCOUNT
  • TAX1
  • TAX2
  • TAX3
  • ANALYTIC ACCOUNT
  • ANALYTIC TAGS

Sample Excel is available inside module.
