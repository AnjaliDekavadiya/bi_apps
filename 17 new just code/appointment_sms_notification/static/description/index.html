Send appointment notifications via SMS in Odoo!!

Now, easily send notifications and reminders to your customers about the progress of their scheduled appointments with Odoo Appointment SMS Notification Management Module. The Odoo admin can choose to send notifications to customers once an appointment request is placed in Odoo successfully and change the status of the appointment request status.

The reminders can also be sent to the customers regarding their upcoming appointment date. The admin can set the days before an appointment date to send the reminder SMS notification to the customer.

Odoo Appointment SMS Notification Module is independently dependent on Odoo Appointment Management System Module and Odoo SMS Notification Module. Both the modules need to be installed first for Odoo Appointment SMS Notification Management Module to work properly.

Detailed Features list
Below is the list of features of Odoo Appointment SMS Notification Management
Send SMS Notifications of Appointment in Odoo
  • Send Notification by SMS to customer on Appointment Creation.
  • Send Notification by SMS to customer whenever the status of appointment changes.
  • Send Notification by SMS to customer as a reminder before the appointment.
Configure the appointment notifications in Odoo
  • Admin can configure whether to send SMS on appointment creation or not.
  • Admin can configure whether to send SMS on appointment status update or not.
Configure the reminder notification in Odoo
  • Admin can configure whether to send an SMS reminder before the appointment or not.
  • Admin can also configure appointment wise whether to send an SMS reminder before the appointment or not
Set time of the reminder of appointment in Odoo
  • Admin can set the time in days or hours before which reminder will be sent to the customer.
  • Admin can configure and update the SMS template according to the need.
Admin can configure SMS Notification Settings for customer.

Admin can update the SMS Notification template also.

Admin can configure the SMS reminder settings and the time duration for each appointment.
