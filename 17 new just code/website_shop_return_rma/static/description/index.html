Website RMA with Return Orders Management

Return Merchandise Authorization (RMA) Management Odoo App

This app allows your customer(s) to create return RMA orders from your website customer portal and allows your sales team to manage return merchandise authorization (RMA) requests by customers as shown in the below screenshots.

An app also allows your customer to view return order(s) from the portal of your website.

Main Features:
  • Customers can create RMA return orders for a product by logging into the Portal of your website.
  • Customers can go to sales order and click the "Return" button to return an item/product that will generate a return order (RMA Order) in the backend side which then be managed by your sales team.
  • Customers can see the list of RMA orders in the portal of your website after submitting a request.
  • Allow your customer to specify the return reason while submitting a request for RMA for an item they want to return.
  • System adds workflow on return order in the back-end: (Draft->Confirm->Approve->Return) - So when new return order request is generated then system will put it on draft/new stage.
  • Allow your inventory team to do return pickings from the delivery orders form (Odoo Standard Feature). It allows only a single product/item on an RMA order so if you want to allow your customer to request for multiple sales order line return cases and then wants to return RMA in the backend with multiple items/products then it will need multiple RMA. If you want multiple lines link on single RMA Order (Return Order) you can check our app https://apps.odoo.com/apps/modules/17.0/website_request_return_rma_odoo/
  • System link-up of RMA order with Sales order when return order request created from the website by a customer from sales order form using return button.
  • On the customer form, a smart button shows all return RMA orders related to that customer.
  • Print RMA PDF Report - Print Return Report as shown sample below screenshot.
  • For more details please watch the video and check the below screenshots.
