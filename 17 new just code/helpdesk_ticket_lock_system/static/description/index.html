Helpdesk Support Ticket Locking System

Support Ticket Lock in Helpdesk Odoo App

APP COMPATIBLE WITH ODOO COMMUNITY EDITION

This app allows support users / technicians / support managers to lock the helpdesk support ticket they are working on.


Main Features:

  • Lock the Helpdesk ticket anytime for a given start and end datetime.
  • Unlock feature is also available so users can unlock tickets when they want.
  • Users who have locked the ticket can update values on the ticket.
  • Other users will get a warning message if they try to update values on the locked tickets.
  • For more details please watch the video and check below screenshots.
