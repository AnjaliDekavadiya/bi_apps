Fleet Repair Integration with Stock Inventory

Fleet Repair and Fleet Repair Tasks Integration with Stock Inventory - Odoo Module

This app allows your fleet repair team and inventory team to integrate Stock Moves and Stock Pickings of Fleet Repair Requests and Fleet Tasks.

Main Features:

  • Integrate Fleet Repair Request with Stock Moves of Stock Pickings. Allow your user to select Fleet repair requests on stock move of picking.
  • Integrate Tasks with Stock Moves and Stock Pickings. Allow your user to select Fleet Task on stock move or picking.
  • Inventory User(s) can select Fleet Repair Requests and Tasks on Stock Pickings Form.
  • Fleet Repair Request Form View with Stock Moves and Stock Pickings Smart Buttons.
  • Task Form View with Stock Moves and Stock Pickings Smart Buttons.
  • Your inventory team can choose Fleet repair requests and Task on stock picking form during incoming, outgoing, internal transfer. And based on this selection stock will be integrated with Fleet repair requests and Tasks of Fleet repairs.
  • User(s) can see Stock Moves by Fleet Repair Requests Report under Stock Reporting Menu.
  • User(s) can see Stock Moves by Task Report under Stock Reporting Menu.
  • For more details about Fleet repair module please click on this link : (https://apps.odoo.com/apps/modules/15.0/fleet_repair_request_management)
  • For more details please see below screenshots and watch the video.
