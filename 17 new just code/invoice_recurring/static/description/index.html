Recurring Invoices

In light of our expanding business, it is essential to streamline our operations and introduce automation. This module is designed to assist accounting users in automatically generating invoices based on predefined time intervals. By doing so, it enhances cash flow, expedites payment processing, reduces the time spent on manually creating individual invoices, and automates the payment workflow. Furthermore, it contributes to fostering stronger customer relationships by scheduling invoice creation on an early basis, ensuring timely and efficient invoicing.

Backend User Capabilities

Backend users can create recurring invoices.

Flexible Recurring Duration

Users can set a flexible recurring duration for the invoices.

Automatic Invoice Creation

Recurring invoices will be automatically generated based on the configured duration.

Cancellation Flexibility

Users have the option to cancel a recurring invoice at any time.

Centralized Summary

A centralized summary of all recurring invoices is available in one place for easy management.

Customer Notifications

Customers will receive notifications two days before the defined recurring invoice date to ensure they are informed and prepared.

Make Recurring Invoice

To Invoice Make a Recurring, go to Invoicing => Customers => Invoices => In action menu click on "Make Recurring" action for generate recurring invoice.
Recurring Wizard

By clicking on "Make Recurring" option, wizard will pop-up which has following functionality.

Fields Description

Name: Name is a way to know which Recurring invoice it applies towards.
Partner: Define Customer name.
Recurring Interval: Multiply with recurring interval like if we give value '2' system will create recurring invoice every 2 months.
Recurring Number: How many recurring invoice want to create by a user.
Recurring Unit: Option is available like Days, Weeks, Months.
First Date: Recurring start Date.
Recurring Invoices

In recurring invoices menu show all the recurring invoices history.
Reminder Mail

Customer will be notified prior 2 days by defined recurring invoice date.

Frequently Asked Questions

How does the system handle recurring invoices' creation?

The system automatically generates recurring invoices based on the flexible duration configured by the user.

Can a user cancel a recurring invoice, and if so, at what point?

Yes, users have the flexibility to cancel a recurring invoice at any time, providing them with control over their invoicing process.

How are customers notified about impending recurring invoices?

Customers receive notifications two days prior to the scheduled recurring invoice date, ensuring they have ample time to prepare and make payments.
