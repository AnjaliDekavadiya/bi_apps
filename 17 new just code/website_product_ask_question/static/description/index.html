Ask Question on Website Shop by Customers / Guests

Ask Question about Product on the Website Shop by Customers / Guests

This app allows customers/guests/visitors of your website to ask questions about products to the seller (your company) from the website shop.


Main Features:

  • Customers / Guests can ask questions about the product before buying the product from a website shop. So if they have any questions before buying then they can send questions to you related to the product and it will create CRM Lead/Pipeline in the backend.
  • The system will create CRM Lead/Pipeline in the backend when customers send questions from the website.
  • For more details please check the video and check the below screenshots.

Available Website Menus:

  • Ask Question
