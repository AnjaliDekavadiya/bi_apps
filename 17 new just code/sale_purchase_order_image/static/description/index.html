Product Image on Sales and Purchase Odoo PDF Reports

This module allows to add image and serial number on sale order line and purchase order line and also on sale report and purchase report.


Main Features:

  • This module allows to add image and serial numbers on the sales order line and also on the Sale Quotation and Sale Order report.
  • This module allows adding image and serial number on purchase order line and also on Purchase RFQ and Purchase Order report.
  • This module allows users to print product image and serial number on sale order and purchase order reports of Odoo standard.
  • Sales settings apply checkbox "Print Line Number and Print Product Image".
  • Purchase settings apply checkbox "Purchase Print Line Number and Purchase Print Product Image".
  • For more details please check below screenshots and watch the video.

Configure On Sales Settings

To display image and serial numbers on order line in report the configuration is available in sale setting.

Configure On Purchase Settings

To display image and serial numbers on order line in report the configuration is available in purchase setting.

Checkboxes on Sales Order Form

You will find two boolean field on Sales order form and based on that report will be printed.

Checkboxes on Purchase Order Form

You will find two boolean field on Purchase order form and based on that report will be printed.
