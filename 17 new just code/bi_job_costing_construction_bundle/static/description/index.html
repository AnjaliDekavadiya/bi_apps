All in One Project Job Costing Bundle (Contracting / Construction Management / Progress Billing / Estimates) Odoo Apps

All in One Project Contracting and Construction job costing bundle Odoo apps helps to manage complete process of the project construction management for costing and material planning process. Basically its helps to manage project job cost and job cost sheet with material request in construction management system in Odoo This Odoo module also help to calculate progress costing and cost calculation in each Construction project job cost with job order with Material Product Purchase Requisitions it use for product purchase requisition process by employee and users. Once purchase requisition is placed for required goods and material, by default it will be placed from the configured location for users and employees. Project Job cost sheet calculates complete cost of each project including material, labour and overhead cost for material it also generates stock movement when material is consumed for the process. Users are able to print different reports for this job costing apps.

Construction bundle has lots of features for job costing management such as manage cost estimations for material labour and overheads, budget management for construction job costing, change request management for construction, contracting and equipment MRO for construction, manage construction projects from website, customer progress billing for project of construction,construction equipment and machine repair , machine and equipment maintenance request for construction, Construction job order and job order integration with expense, Job order quality inspection and instruction, job contracting issue tracking portal, Construction document portal for project contracting, work package for construction, Project job costing contracting dashboard, job costing sales estimates, construction vendor subcontractors, website job work order and work order inspection, Work order start and stop timer with digital signature, transmittal and Submittals Document for Construction Job Costing, Job Drawing and Job Drawing image for construction and Timesheet management for project job costing with website and mobile portal view etc. .

Features

Job Order

Work Package creation by adding multiple job order / work order to include in it. Link Project and Job order with Maintenance request & Equipment. Customers/Visitor/Internal Employees can create job order or work order from website.

Email Notification

Send mail from the attachments with the attachment. Work Package send by email feature.Employee and/ Technician can send message & attachment and also view message history. Instance Email Notification to customer for created job order.

Issues

Your Customer/Construction Team can send issues request from your website. Customer/Construction Team can check issue requests on My Account page. Manage Project Issue for project.

Expense

Employees who is working on job order they can submit expense rgarding that job order with pictures of expense.Automatic manage your job cost sheet budgets.

Separate Page

Every image/Document will open in separate page. Document download from portal Allow to download the document from portal view.

Create Repair Request from Job/Work Order.

You can create multiple repair request from job/work order. Repair order and job order are combined, Smart button on job order to access repair order.

Print Reports

Print PDF report of requested Issue. Work Package PDF report Customer can have portal in My Account to view and print job order they have created. Users are allowed to see report of each purchase requisition including other informations. PDF reports for Job Cost Sheet and Job Orders.

Waste Management

Allow job order user to add Waste Materials on Job Order form and based on that Create Internal Picking or Stock Scrap based on Waste Method selected.

Add photo/Image in Job Order

This modules allows you to add Photo/Image/Document in Job Order.

Signature

Digital signataure will be store in backend job order record.

Manage meeting minutes become easy

User can manage the meeting minutes of project/comstruction easily. Define meeting agendas, action items and conclusion.

Timesheet

Project issue User/Technician can fill timesheet from issue. User can fill timesheet by website.

Timer

Allow your Employee to start/stop/pause clock timer on job order form of website portal my account page.

Dashboard By Project

Display dashboard by Project. Filtered view for job/cost-sheet/labour/overhead/timesheet/materials.

Document download from portal

Allow to download the document from portal view.

The directory menu is only appear to the manager

The menu directory is only visible to the document manager.
