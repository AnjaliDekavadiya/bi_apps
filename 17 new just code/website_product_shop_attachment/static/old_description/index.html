Documents/Attachments on Website Shop

This module add feature on E-commerce/Eshop to allow customers to check and download product documents. For example company can allow customer to download documents like Product detail specification, Terms and condition, etc...

Admin can attach document/specification by product. Admin can show/ hide feature using customize option on website.

Customer/Portal Users/Visitors/Guest can download documents from product page on website.

Attachments/Documents on Product Template View

Sales tab on product form where user has to set all documents which can be downloaded by customers..

Product Cart/Page on Shop

All documents attached in product in backend will be available on shop and can be downloaded by customers.
