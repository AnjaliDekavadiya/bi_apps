Royal Mail Pro Shipping

Upscale Shipping Convenience in Odoo!

Logistics delivery is made easier with the help of Royal Mail Pro Shipping. Create several deliveries of products and a combined order for them for Royal Mail on Odoo. Each day, the cron scheduler generates combined orders for Royal Mail. It allows both domestic and international shipping. Admin can set rates for shipping as well.

Technical Requirements for Royal Mail Pro Shipping API Integration

  • Provider: Royal Mail
  • Division / API: PRO SHIPPING API (v4.0-RM)
  • Region: United Kingdom
  • Used for: Shipping, Tracking
  • Get API access: https://www.royalmail.com/business/parcels/shipping-solutions/pro-shipping
    • Please ensure that you have a valid Royal Mail Pro API key and Secret to integrate.

Why use Royal Mail Pro Shipping?

Managing deliveries can be complex, involving a lot of tracking and organization. To simplify this, you can use a carrier service to handle these tasks. Royal Mail is a top-quality delivery company known worldwide. It can assist in getting orders to customers promptly and accurately. Our module connects your Odoo system with Royal Mail to create an efficient delivery system. With the Royal Mail Delivery Carrier module, you can easily prepare packages for delivery. Features like listing items for delivery, handing them over to Royal Mail for shipping, and tracking them are also included.

Advantages of Pro Shipping

It offers an efficient solution for online retailers. It allows the creation of shipping labels with ease, even across multiple workstations. Businesses can save time and effort by sharing a single account among colleagues. It also prevents duplication. The Pro Shipping API simplifies paperwork and accounting by automatically adding multiple orders from different sources to your account. It has customizable file templates for bulk order uploads. Hence, it aligns perfectly with business needs.

Using Royal Mail Pro Shipping in Odoo

  • Place orders with delivery from both the front and back end.
  • Configure delivery options for both domestic and international orders.

Generating Shipments in Odoo

  • Automatic creation of shipping labels and tracking information after validation.
  • Shipments are generated after the order is validated.

Set Default Shipment Value

  • The system will automatically use these default values when creating packages.
  • Set Default packaging options for both domestic and international packages.

Combined Order with Tracking

  • Generate an invoice for Royal Mail for each package daily.
  • Package tracking is accessible from the backend for admins and via a tracking link for customers.
