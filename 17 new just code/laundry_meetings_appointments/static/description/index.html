Laundry Appointments for Collection and Delivery

Laundry Meeting Appointments in Odoo

Collection of Clothes and Delivery of Clothes using Appointment from Laundry Request

This app allows you to manage laundry collections and delivery appointments with your customers using odoo calendar meetings app.



Main Features :
  • Allow you to set the Collection and Delivery User on laundry requests.
  • Laundry User / Manager can create Laundry Appointments for Collection and Delivery.
  • When clicking on the Create Appointment Button on the laundry request system will open the wizard with select type Collection appointment, Creating Collection Appointment will set the Collection User on Wizard Collection / Delivery User Field automatically.
  • When clicking on the Create Appointment Button on the laundry request system will open the wizard with select type Delivery appointment, Creating Delivery Appointment will set the Delivery User on Wizard Collection / Delivery User Field automatically.
  • Showing the created collections and delivery Appointments from the laundry requests.
  • My Collections Appointments for collection persons.
  • My Delivery Appointments for delivery people.
  • Collection and Delivery group is created so you have to create a user for collection person and delivery person using that group which will allow them to log in and check laundry requests where they are responsible and also allow them to view appointments meeting related to that laundry request.
  • For more details you can watch the video and see the screenshots below.
