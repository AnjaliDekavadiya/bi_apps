Fleet / Vehicle Repair Request Management

Fleet/Auto/Vehicle Repair Service Scheduler, Online Request Management, and Fleet Integration

This app allows you to manage online requests from your customer for fleet/vehicle/service repairs and it also integrates with Odoo's standard fleet app.
This module is fully integrated with the Odoo fleet module and repair module.


Main Features:

  • Your customers can send Fleet/Auto/Vehicle repair requests from your website and also attach documents/images as attachments while submitting a form.
  • Generation of unique repair request on submission and record it as fleet repair request in the backend.
  • Customers can check the status of all their own fleet repair requests submitted by them on the account page of your portal.
  • Allow your user to print a PDF report of fleet repair requests.
  • Repair User / Technician can communicate with customers using chatter and fill timesheets on workings on their requests.
  • The repair Manager can close fleet repair requests and send repair requests for feedback to customers.
  • Customers can give feedback and rating of fleet repair requests done by the user.
  • Manage your fleet repair request using the assignment repair team.
  • For more details please watch the video and check the below screenshots.

General Workflow:

  • Customers come on websites and want to book fleet/auto/vehicle repair requests.
  • Customers go to the link to create a fleet/vehicle repair request.
  • Customers can see the form and fill in details. (view timeslots button allows him to see available time to meet advisor/representative).
  • Customer can choose multiple services - Pricing is shown if price is not available on the service form in the backend then it will show 'Ask Advisor for Pricing'
  • Customers can choose Year, Make, Model, and enter a number plate / License Plate. (System will search License Plate in odoo list of fleets/vehicle and if found then it automatically link Vehicle in backend otherwise repair manager has to create a new vehicle.).
  • On submission of repair request system will create repair request in the backend as well as create an appointment/meeting with an advisor.
  • The repair manager can open a repair request in the backend, see details entered by the customer and also fill some details manually and assign that request to technical or repair use.
  • The repair manager will create the vehicle (if not available in the system), service logs, and repair order.
  • Repair users can work on repair requests and fill in timesheets.
  • Repair managers can close the request and will send an email to the customer to request feedback.
  • Customers can fill in online feedback after getting service.

Role of customer

  • Customers can go to the link to create a fleet repair request from your website.
  • Customers can send Fleet/Auto/Vehicle repair requests from your website by filling out a form and also attach documents/images as attachments while submitting a form.
  • Customers can fill in online feedback using a link in the mail.
  • Customers can view submitted requests in my account portal.

    • Role of repair manager

  • Manager can assign requests to repair Users/Technicians.
  • The manager can create a meeting appointment with the team and customer.
  • Managers can create a task/job orders
  • Generation of unique repair request on submission and record it as fleet repair request in the backend.
  • The repair manager will create the vehicle (if not available in the system), service logs, and repair order.
  • Repair Manager can view and manage vehicles using buttons.
  • The repair manager can create and manage a work order/tasks related to request.
  • A print pdf report of the repair request.
  • The repair manager can close the repair request at the end of the repair.
  • Repair managers can close the request and will send an email to the customer to request feedback.
  • View feedback rating repair request form on the backend.

    • Role of Repair User/Technician

  • Repair users can see and manage repair requests assigned by the repair manager.
  • They can work on repair tasks/job orders and repair orders assigned to them. And fill in timesheets.
  • Repair User/Technician can communicate with customers using chatter and fill timesheets on workings on their requests.
  • Repair User / Technician can also view meetings/appointments on request, and vehicle services since they are going to attend it.
