This app allows your workflow on sales orders for triple approval levels based on configuration of amounts and limitation of approval.

Approval Sales Order Triple: Department/Sales Manager -> Finance Manager -> Director Approval.

System is flexible to have approval buttons based on amounts configured on company form. For example if sales amount less 5000 then only required single level approval but if amount of purchase order is greater than 5000 and less 8000 then it will need Sales manager + Finance manager approval.

Main Features:
  • Allow you to configure Finance Approval Start Amount Range, Stop Amount Range.
  • Allow you to configure Director Approval Start Amount Range, Stop Amount Range.
