Help Desk Service Level Agreement

This app allow your helpdesk team to have service level agreement(SLA) on Help Desk.

Main Features:

- Allow to you to setup SLA Levels under configuration.

- Allow you to link SLA level on your Customer form.

- Calculating Deadline of tickets based on SLA level configuration. System allow you to setup SLA level by selecting Category of ticket and priority based on that you can configure number of Hours/Day/Week to have automatic Deadline on Tickets.

- Helpdesk SLA Team: You can setup support team with their Working Time/Calendar and setup Estimated Response Time / Reply Time / Working Time for that stage.

- View SLA time logs for that ticket on helpdesk form based on Helpdesk SLA Team setup.

- Helpdesk SLA Analysis to anlysis your SLA logs by ticket numbers.

- For more details you can contact us or watch Video.

* Only compitible for Odoo Community Edition of Odoo.

System Created New Menus:

  • Helpdesk/SLA/Helpdesk SLA Levels
  • Helpdesk/SLA/Helpdesk SLA Analysis
  • Helpdesk/SLA/Helpdesk SLA Team

Helpdesk SLA Levels -List View

You can configure multiple levels for your SLA and which will be set on Customer form and based on that ticket will be having its deadline.

Helpdesk SLA Level -Form View

This allow you to configure difference types category of tickets by priority based on that you can setup Hour/Days/Week which will be using by Tickets to set Deadline Date on ticket form.

Helpdesk SLA Team -List View

Helpdesk SLA Team View

This configuration allow you to setup SLA timings for given Support Team by Calendar. This will create SLA Logs on helpdesk while team/techincian/responsible moving and working on ticket stages.
