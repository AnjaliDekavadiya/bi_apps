Upsell on Sales Subscription / Sales Contract

Upsell Management in Odoo App

Customer Sales Subscription / Contract Upsell in Odoo Module

Only Compatible for Odoo Community Edition

This module allows your sales team to do Upsell on Sales Subscription / Sales Contract.


Main Features:

  • Allow the sales team to upsell on Customer Contract / Sales Subscription.
  • Options during upsell are: You can directly update contract lines / create new contract lines or you can create new sales quotations.
  • You can view upsell Contract History on Contract.
  • You can view Sales Order on Contract.
  • Option 1
    • Update existing contract/subscription line while doing upsell if product matches with existing contract/subscription line.
  • Option 2
    • System will create new contract/subscription line if product does not match in existing contract/subscription lines.
  • Option 3
    • Create Sales Quote from upsell and then once sales order confirmation system will append/modify contract/subscription lines.
  • For more details please see video and check below screenshots.
  • Only for odoo community edition.
