Project / Analytic Account Profit and Loss Report Odoo App

Profit and Loss for Project / Analytic Account

This app allows you to print a Profit and Loss report for selected analytic accounts/projects as shown below screenshots.


Main Features
  • Allow you to print a profit and loss report in pdf format for analytic accounts and projects. This report is built based on analytic items / analytic entries in Odoo which come from various sources like customer invoicess, vendor bill, timesheets etcâ€¦
  • You can select the start and end period on the wizard and analytic accounts and the system will print Profit and loss reports for that as shown in the below sample.
  • Please note that the Cost/Revenue button on the analytic account form has been renamed now to Gross Margin since version 15.0. So if you see *Cost/Revenue* button on any below screenshot(s) and in the video please consider it as a new renamed button *Gross Margin*.
  • For more details please check the below screenshots and watch the video.

Menus Available:

  • Invoicing
    • Reporting
      • Analytic Profit and Loss

Important Note for App: For Analytic entries (Analytic Items) where Financial Account is set system will show it on the Profit and Loss report as Financial Account name but for Analytic Entries where Financial Account is not available on analytic lines then the system will show it as Other Income or Other expense (For e.g. Timesheet Entries does not have a financial account set on its analytic lines.).
