Bugs and Issue Management Odoo App

Project Task Issue and Bug Management App

This app allows your project team to create an issue for a task of a project. Allow your project team to manage issues and bugs of projects and tasks.

Main Features:

  • Allow your project team to create an issue for a project task in the backend.
  • After creating an issue assigned to the project user who can work on that issue as shown.
  • Allow your customers to view bugs/issues on the portal of your website.
  • Allow your customers to report bugs/issues directly from the task view on the portal of my account on your website.
  • Allow you to create and manage issues types under configuration.
  • For more details please check the below screenshots and watch the video.
