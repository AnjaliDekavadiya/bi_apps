Agriculture Farm Manage with Material Requisition Odoo App

Crop Request with Material Requisition Integration

This module allows your employees / users to create material requisitions for agriculture crop requests.


Main Features:
  • Allow you to have feature of agriculture with material requisition.
  • Agriculture/Material Requisition/Material Requisitions : Allow your employees to make material requisitions for crops.
  • Employees can request multiple materials / items using Material Requisition requests.
  • Approval of department head of requisition request from user.
  • Approval of manager of requisition request from user.
  • Request for Material Requisition will go to stock/warehouse as internal picking / internal order and purchase order as shown.
  • Warehouses can dispatch material to employee locations and if material is not present then procurement will be created by Odoo standard stock module.
  • Material Requisition users/officers can decide whether a product requested by an employee will come from stock/warehouse directly or it needs to be purchased from a vendor so we have a field on requisition lines where responsible can select Requisition action: 1. Purchase Order 2. Internal Picking. If option 1 is selected then the system will create an internal order / internal picking request and if option 2 is selected the system will create multiple purchase orders / RFQ to vendors selected on lines.
  • For more details please check below screenshots and watch the video.
