Odoo Excel Connector

Connect your Odoo with Excel.

Odoo Excel Connector integrates your Odoo to Excel. It allows syncing data from Odoo to Excel or LibreOffice. You can configure the details you want to export in Excel and set the frequency. The spreadsheet can help you build reports and manage data of any module (accounting, sales report, customer, etc.).

Synchronizing Data between Excel and Odoo

The module allows you to sync data between the platforms and set the frequency for updates from MS Excel or LibreOffice. It generates an ODC file and a URL to access the Odoo. It creates a URL or ODC file that, when opened in MS Excel, will fetch the data according to the defined conditions.

Reduce Manual Effort in Exporting!

The Odoo app allows you to create a template for exporting data in an Excel spreadsheet that can be executed at a set duration to export data from Odoo. Moreover, you can make multiple templates, execute them, and manage their active status.

Secure Odoo Excel Connector

The module generates an access token and URL that can only be accessed when the export template is in an Active state. Also, you can check the access history of every connection. If you find any suspicious IPs, you can renew the access token, and this will also change the URL so only the user with a new URL and access token can use it. You can also filter out the data based on your need. Additionally, if the export template is in a draft or inactive state, the URL or access token will work to export data.
