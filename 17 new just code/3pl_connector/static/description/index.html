Odoo Third-party logistics Connector

Third-party logistics (3PL) or TPL, is a strategic business practice where organizations leverage external entities to outsource specific aspects of their distribution, warehousing, and fulfillment services. These external partners, known as third-party logistics providers, specialize in integrated operations that encompass both warehousing and transportation services


The Odoo 3PL connector works as a bridge for import and export operations through XLSX or CSV file between Odoo and third party logistic system.

Please Note:
1.To import stock of the product, users have to manually export the product file and then they can update the product quantity in that file and then use that file for import stock operations
2.While using Lot wise/ Serial number products, it will be recommended to use manual stock import instead of auto inventory import

Import Operations

User can select individual or multiple import operations at once.
Notes:
1. While import operations of Sales Transfer, Sales Return Transfer and Purchase Transfer it will be executed in existing records, it will not create any new record.
2. For Import stock operation, Stock will be updated in shown location.

Export Products
