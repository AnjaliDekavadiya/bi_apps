Project Task Deadline Reminder Email

This Odoo module sends reminder email to the responsible user of the project task.


Main Features:

  • This module sends an auto reminder to the responsible user of the task if the deadline date = Today date.

  • Cron job will run everyday and search for tasks which are due today and send reminder emails to user of the task.

  • Custom Email template available with name "Project Task Reminder Email" so you can change email content.

  • For more details please see below screenshots and watch the video.

Edition Compatibility:


Enterprise Odoo.sh Community

Project Task Form - Configuration of Task Reminder

If set this box then only this task will be consider for reminder.

Automation Scheduled Action - Cron Job

Email to User/Employee

View Now button will allow user to jump to related task directly. This email will group all tasks which are deadline today for that user and send summary table to user/employee by email.
