Odoo Age Verification

Now, Restrict Underage users to enter your Odoo Website!

Odoo Age Verification acts as a gate to allow only the users above the pre-mentioned age group. You can also add a message for the users who are unable to login because of the Odoo Age Verification.

Sometimes the content or products you add on your website may not be appropriate for all audiences. In this case, you can add a statutory message for your website visitors so that only the audience above the mentioned age can visit the website.

However, just a message won't stop the audience that doesn't fulfill the criteria of entering your website. Adding an age gate is the perfect way to allow only the audience above the age limit you desire to your website.

Odoo Age Verification allows you to add an age gate to your website so that only people of a restricted age group can enter your website. You can set the age bar from the Odoo Backend to restrict the mentioned age and below it.

Set the Age Verification gate in your Odoo Website as per your flexibility

If you want to add an age restriction gate in your Odoo Website then Odoo Age Verification is developed for you only. Configure the restrictions you wish to enact on your website in the Odoo backend and the changes will be reflected in the Odoo website frontend.

You can add customized images as well as messages on the age verification popup to make it more user-friendly. You can configure the age verification feature uniquely to all your websites as the module is compatible for Multiple Websites.

You can also set the age verification gate for all over website or particular web pages. If you select the "Page Wise" option then there you need to add web pages routes.

You can also ask visitors to add their date of birth so that the eligibility of the visitor can be calculated. Or, you can simply add a checkbox for the visitors to verify their eligibility. Now, if the visitor marks it as ticked then he/she can visit the website otherwise not.
