Offers Coupons and Vouchers to your Odoo Website Customers

Website Coupons & Vouchers come up with the functionality to create digital discount coupons for customers in Odoo. You can set the discount as fixed or a percentage of the total cart value.
Each coupon contains a unique code to validate it with the purchase. Customers just have to enter the available coupon code to get the defined discount. The Odoo user can create general vouchers and coupons for all customers, or special coupons for specific customers.

Introduces the Discount Coupons to Odoo Website

  • It allows you to create and manage website coupons.
  • Every coupon includes a unique code.
  • Customers need to use the code at the checkout page to avail of the discount.

Create Coupons for Selected Products

  • Create coupon codes for selected products.
  • Or, create codes for some specific products..

Specify the Minimum Cart Amount Eligibility for Coupon Use

  • You can choose to enable minimum cart amount validation for a coupon.
  • Specify the minimum cart amount for customers to avail of the coupon discount.
  • Every Odoo website coupon can have a different minimum cart value.

Limit Coupon and Voucher Validity and Usage

  • The Odoo app allows entering the number of times a coupon can be used on the website.
  • Once the number is reached the coupons will automatically expire.

Shows Available Coupons on Website Checkout Page

  • Customers can see the available coupons on the product page.
  • Customers can redeem the coupon from the checkout page.

View Usage History of Website Coupons in Odoo

  • You can view the usage history of all the coupons.
  • After reaching the defined coupon limit, they automatically expire.
