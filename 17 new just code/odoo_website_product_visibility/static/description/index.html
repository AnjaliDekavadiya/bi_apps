Hide / Show Product Category on Ecommerce in Odoo

This app allows you to set visibility of your shop products and product categories on a website shop for guests and logged in customers.



Main Features:
  • Allow limited product categories and products to show to your customers.
  • Allow limited product categories and products to show for public / guest visitors.
  • Allow you to filter products and categories on website shop page in Odoo.
  • Allow a set list of categories and products to show for customers on customer form.
  • Allow a set list of categories and products to show for public/guest visitors on the menu 'Public Guest Visibility'.
