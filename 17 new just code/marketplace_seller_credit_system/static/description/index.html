Odoo Marketplace Seller Credit Management System

Attract Buyers By Rewarding Them With Credit Points On Every Purchase!!

Odoo Marketplace Seller Credit Management System module allows the seller to reward credit points to their customers according to some defined base rules. The earned credit points can be redeemed by the customers either on the cart page or at the checkout page.

Set Up Rules for Customers To Earn Credit Points

The module makes it simple to set up rules for customers to earn credit points on the Odoo Website so that they can get the most out of their experience.

Create the Base Rules according to which the customers get credit points while shopping from your website. A seller can set the value for each credit point and create different credit rules based on - Purchase Amount, product price, or on a specific product.

Sellers can reward credit points to the customer, on the confirmation of the order or on the completion of the order according to the configuration from the backend. Details about the credit point transactions can be viewed by the customers and the store admin as well.

Detailed Features List

Introduced Credit Point System in Odoo Marketplace

  • The module lets you introduce a credit point system for your customers.
  • The registered customers earn credit points for making purchases on your Odoo Website which can then be redeemed on subsequent purchases.

Set the criteria for Credit Points

  • Credit points can be awarded to the customer by configuring it in the Odoo backend.
  • The Odoo website admin can set a credit base for the customers to earn credit points based on purchase amount.
  • Credit Base is created on the basis of Product Specific, Purchase Amount and Product Price.
  • Sellers can create multiple credit base.

Set the Threshold to earn Credit Points

  • The threshold purchase amount can also be defined for the customers to be eligible to earn credit points by the seller.
  • Customers need to sign up to earn credit points. After that, on every purchase customer gets the credit points according to the defined credit base.

Set the Credit Point Redemption Criteria

  • The points can be redeemed on a one-time basis or partially on multiple orders as configured by the Odoo admin in the backend.
  • Set the amount that can be redeemed with each credit point.
  • The admin can also set the maximum purchase amount that can be redeemed by the customer using credit points in a single order on Odoo Website.

Check the history of credit points

  • The history of credit points redemption can be seen by customers, sellers and the store admin in the Odoo backend.
  • Moreover, the customer can also see the credit points in his account and their redemption value on the checkout page.

Show the Credit Point System Information in frontend

  • The information about the current credit points running on the Odoo website and its benefits are displayed on the customers checkout page.
