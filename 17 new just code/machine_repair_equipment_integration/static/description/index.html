Machine Repair App & Equipment Maintenance Integration

Integration of Machine Repair and Equipment Maintenance - Odoo App

Machines and Equipments

This app allows you to create equipment from machine repair request form as shown below screenshots.

Main purpose of the module is to link machines with equipment so you can create equipment of your customer machine and keep record of that equipment in your equipment list and at the same time odoo will generate maintenance requests periodically so that you can follow up to your customer about maintenance of your machine / equipment.

You can view an equipment smart button on a machine repair request. Now when customers send machine repair requests again in future then you do not need to create equipment again but you can choose it on machine repair form. So you will maintain a machine list of your customers in odoo as equipment and keep track of it.

Same time you know in machine repair module machine is product in odoo as well. So in summary the system allows you to create equipment from machine repair requests and then equipment will be listed in odoo equipment menu and it will be then using maintenance preventive flow it will create maintenance requests.

Main Features:

  • Repair Manager can create a Equipment for a Machine Repair request:-Repair Manager can create equipment for a machine repair ticket using the Create Equipment button. And also Repair Manager and Repair User can see created equipment on a machine repair ticket.
  • Repair Manager can see a Machine Repair for a Maintenance Equipment:- When clicking the machine repair button, Repair Manager can show a created machine repair ticket of maintenance equipment.
  • Only Machine Repair Manager can Create Equipment:-Only Machine Repair manager see the Create Equipment button on a Machine Repair same time Repair user only can see equipment but he/she is not able to create.
  • Users can see all Equipments through Equipment Menu:-Show All Equipments on the Equipment Menu and View all Equipments.
  • Also Repair Manager and Repair User can create and view all Maintenance Requests through Maintenance Request Menu:-Click the Maintenance Request menu and view and create all Maintenance Requests. Same time odoo also creates Maintenance requests periodically based on configuration on equipment.
