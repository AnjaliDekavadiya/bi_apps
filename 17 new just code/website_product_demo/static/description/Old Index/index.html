Request Demo Website Shop/Ecommerce

This module allow user to send request for product demo through website shop/ecommerce.
Customer can request demo from website shop and that will create lead in Odoo backend. - So Request Demo linked with Odoo CRM Lead
This app will work for logged in customer and also work for your visitor of your website.

Request Demo Button on Shop

Request Demo Button - Click Open Form

After Sending Request - Thank You Message

Confirmation Email to Customer

Odoo Lead - Created

This allow your sales team to track and work on it for demo.

Request Demo Button on Shop - for Visitors

If visitor is not login in Odoo website but still they can request demo by entering name and email on wizard.

Odoo Lead - Created
