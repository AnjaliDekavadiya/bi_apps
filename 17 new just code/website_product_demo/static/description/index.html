Request Demo for Product from Website Shop Odoo App

This app allows your customer/guest user to send a request form demo from your website shop product page as shown the below screenshots.


Main Features:

  • Customers can request a demo of the product from the website of the shop.

  • This requested demo creates a CRM pipeline/opportunity in the backend under the CRM application of Odoo standard.

  • The visitor of your website without logged in also requests a demo by adding their details on the demo request form.

  • Acknowledgment sent by the sales team to the customer when the request is received in the CRM.

  • For more details please check the video and check the below screenshots.
