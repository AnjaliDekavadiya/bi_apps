Multi Branch Odoo Enterprise Accounting Reporting

Only for Odoo Enterprise Edition

This app allow you to filter enterprise accounting reports by branch filter.

Main Features:

  • This app allows you to below listed accounting reports by branches.
    • Statement Reports
      • Balance Sheet
      • Profit and Loss
      • Cash Flow Statement
      • Executive Summary
      • Tax Report
    • Audit Reports
      • General Ledger
      • Trial Balance
      • Journals Report
    • Partner Reports
      • Partner Ledger
      • Aged Receivable
      • Aged Payable
