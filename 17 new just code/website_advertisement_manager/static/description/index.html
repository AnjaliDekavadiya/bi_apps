Odoo Website Advertisement Manager

Host advertisements on your website in Odoo!!

Odoo Website Advertisement Manager is a module in which several block positions are created on the website and the customers can purchase those blocks for the advertisement purpose and can add desired content. The block content will be published automatically according to the booking date of the block.

Characteristics of this plugin?

  • Advertisement Blocks link will be visible on the website.
  • A user can view all the available block positions with their details.
  • A user can check for the availability of an ad block position.
  • Admin can create and view all the orders of the ad blocks here.
  • Users can click on the add to cart button to book the corresponding Ad-block position.
  • Ad Block is added to the cart with the booking date of the user.
  • Booking of ad blocks for given dates is confirmed after making the payment.
  • Customers can view all their ad-block bookings in the "My Accounts" section.
  • Users can view all the ad blocks booked by the user here.
  • Users can add the content to this ad block from here.
