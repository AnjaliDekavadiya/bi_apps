Product Warranty and Warranty Policies - Ecommerce / EShop in Odoo

Product Warranty Information Show to Product Page of Website in Odoo

This module allows show product warranty and warranty information on the product page of Ecommerce / EShop at the website. You will have to configure it on product form in Odoo backend.

Main Features

  • Configure warranty and warranty details on product template form in Odoo backend and show to your customer on the product page of your website shop.
  • For more details please check below screenshots and watch the video.

Note:

  • Currently (Odoo standard website_sale module) E-Shop showing static content of policies for all products which is shared by all E-Shop products. This module will remove that limitation and allow admin user to write product policies by each product.
