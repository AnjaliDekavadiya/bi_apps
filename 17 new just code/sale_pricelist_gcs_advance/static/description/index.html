Advance Product PriceList (Sales/Website/POS)

We all know the odoo default pricelist is provides good features but not provides advance level features. Which are really needed for the every business to give more and more offers to customers.

What our app provide?

This app allow you to configure advance level features to you can give more and more offers to customer. You can configure like Category along with Brand and also allow to configure Brand along with Category and many more. You can use this app with Sales orders, Website orders, POS orders, ...etc

App Installation

After, Purchase app you need to install our app.

Configure pricelist rule with only Category Level

1. Goto Sales -> Products -> Pricelist -> Create/Open any Pricelist -> Price Rules Tab
2. On pricelist rule select Apply On: Product Category
	>> Product Category: Select product category
	>> Product Brand: Leave product brand
	>> And with this you can configure other option like Min. qty, Start date, End date, Compute Price,...etc
	
	>> When you select product on Sale Order, POS, Website, and Others that time system will check the if Category will match then apply this
	pricelist rule to that product and will apply your offer.

Configure pricelist rule with Brand Level along with Category Level

1. On pricelist rule select Apply On: Product Category
	>> Product Category: Select product category
	>> Product Brand: Select product brand
	>> And with this you can configure other option like Min. qty, Start date, End date, Compute Price,...etc
	
	>> When you select product on Sale Order, POS, Website, and Others that time system will check the if Category and Brand both will match then apply this
	pricelist rule to that product and will apply your offer.

Configure pricelist rule with only Brand Level

1. On pricelist rule select Apply On: Product Brand
	>> Product Brand: Select product Brand
	>> Product Category: Leave empty Product Category
	>> And with this you can configure other option like Min. qty, Start date, End date, Compute Price,...etc
	
	>> When you select product on Sale Order, POS, Website, and Others that time system will check the if Brand will match then apply this
	pricelist rule to that product and will apply your offer.

Configure pricelist rule with Category Level along with Brand Level

1. On pricelist rule select Apply On: Product Brand
	>> Product Brand: Select product Brand
	>> Product Category: Select product Category
	>> And with this you can configure other option like Min. qty, Start date, End date, Compute Price,...etc
	
	>> When you select product on Sale Order, POS, Website, and Others that time system will check the if Category and Brand both will match then apply this
	pricelist rule to that product and will apply your offer.
