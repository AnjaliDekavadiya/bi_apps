Crop Scouting Feature for Agriculture Management

Crop Scouting Odoo App for Farm Manage Agriculture

This app adds the feature of crop scouting for agriculture so using this app you are allowed to create a crop scouting form for crop and crop request and inputs data on it.


Main Features:
  • Agriculture / Scoutings : Allow your user to create crop scouting for crops as shown and input information as shown in below screenshots.
  • Agriculture / Configuration / Soil Temperatures : Allow you to create and manage soil temperatures for crops under configuration.
  • Agriculture / Configuration / Soil Conditions : Allow you to create and manage soil conditions for crops under configuration.
  • Agriculture / Configuration / Winds : Allow you to create and manage winds for crops under configuration.
  • Agriculture / Configuration / Air Temperatures : Allow you to create and manage air temperatures for crops under configuration.
  • Agriculture/Configuration/Cloud Covers : Allow you to create and manage cloud covers for crops under configuration.
  • Agriculture / Configuration / Scouting Insects :Allow you to create and manage scouting Insects for crops under configuration.
  • Agriculture / Configuration / Scouting Diseases : Allow you to create and manage scouting diseases for crops under configuration.
  • Agriculture / Configuration / Scouting Weeds : Allow you to create and manage scouting weeds for crops under configuration.
  • Agriculture / Configuration / Scouting Plant Populations : Allow you to create and manage scouting plant populations for crops under configuration.
  • Agriculture / Crop Requests / Crop Requests : Smart button of scoutings on crop request form view.
  • Agriculture / Crops / Crops : Smart button of scoutings on crops to view scoutings.
  • Agriculture/Scoutings : Allow your user to Print PDF Scouting Report as shown.
  • For more details please check below screenshots and watch the video.
