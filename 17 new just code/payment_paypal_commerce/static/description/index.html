Detailed Features List

Integrate the PayPal Method to your Odoo Marketplace.

  • Facilitates the admin of a store to integrate PayPal Payment Method for their customers.
  • Allows your customers to pay with the secured and safe payment process without any hassle.

Configure the payment method easily

  • You can retrieve the client and the secret key for configuring the payment method
  • The secret key and Client ID is to be added to the module for complete integration of PayPal with your Odoo Marketplace

Allows sellers to integrate PayPal for their customers

  • The module even let your sellers integrate PayPal for their customers.
  • Sellers can integrate only after authenticating with PayPal.

Provides commission based on sellers

  • It allows store admin to receive the commission based on the purchase.
  • You can set different commissions for different sellers.
  • The commission is directly transferred to the admin's account.

Supports Instant and Delayed payment Disbursement Modes

  • You can allow Instant and Delayed payments features (also known as, PayPal Commerce).
  • On choosing Instant, the sellers receive the payment instantly as the customer makes payment and places the order.
  • On choosing Delayed, the amount is kept on hold, you can manually transfer the amount to the seller.

Seller Authentication With PayPal

  • The sellers have the leverage to allow PayPal for their customers to make payments.
  • Sellers can integrate PayPal for their customers only after authenticating with PayPal.

Processing Refunds

  • Easy and safe refund process of payment to the customers
  • Refunds can be processed by you securely and conveniently.
  • Setup & Configure
  • Website View
  • Seller View
  • Admin View

PayPal Commerce Option available on the payment Page

  • PayPal credit option available to pay at checkout
  • PayPal commerce Payment Confirmation

PayPal commerce payment option under seller view

  • Click on connect with paypal button to initiate the PayPal commerce payment.
  • Fill the required details and connect successfully with PayPal.

Choose the payment method, i.e instant or delayed.

  • Instant Payment: Direct customer to seller
  • Delayed Payment: Initially, Customer to admin then release it to the sellers accordingly.
  • You need to register first time using Register Webhook Button to complete configuration

Payment Transactions

  • Keep a track on the transaction done by PayPal payment gateway.
  • Check the transaction of each sale order along with the seller's name.

Direct Payment order line

  • The payment is directly transferred from customers to sellers.

Delayes payment order line

  • You can delay the seller's payement and release it afterwards using release seller payment option

Paypal Credentials

  • As this module provide Paypal Payment API integration thus first of all you will have to create an account on Paypal.
  • Create an app to receive REST API credentials for testing and live transactions.
  • Finally paste the client ID, secret ID, Merchant ID in Paypal Commerce Payment Acquire in Odoo.
