Manufacturing Requisitions for Material

Material Manufacturing Requisition in Odoo App

Employee Material Manufacturing Requisition

This module allows your users to create material requisitions with the create Manufacturing Order option on Requisition Lines.


Main Features:

  • Allow your users to create Requisition requests for products/materials which will be Manufactured/Produced by your company.
  • Third option has been added on Requisition for where users can select Create Manufacturing options.
  • Allow your Manufacturing manager or Requisition manager to choose Bill of material while creating Manufacturing order.
  • For more details please watch the video and check below screenshots.

Menus Available:

  • Manufacturing
    • Operations
      • Requisitions
