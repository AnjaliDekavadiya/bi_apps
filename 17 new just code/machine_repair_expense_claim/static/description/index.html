Machine Repair with Employee/User Expense Claim

Create Expense for Machine Repair Jobs

This app allows your employees to submit expenses for machine repair orders / requests.

System allows your employee to select a machine repair ticket on expense form (Odoo standard expense app) and then your employee(s) can submit an expense to the manager (Odoo standard expense process) and the machine repair ticket reference will appear on the expense sheet as well.

Main Features:

  • Expenses/My Expenses/My Expenses : Machine Repair Request field added so your employee/user can select while creating expenses.
  • The Create Report Button (Odoo standard) of Expenses will move machine repair request field value to the expense sheet.
  • Expenses/My Expenses/My Expense Reports : after creating a report of expenses, the machine repair request field value will be set.
  • Show Smart Button of Expenses on Machine Repair Request Form.
